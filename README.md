# 🚀 Multi-Container Runtime with Kernel Memory Monitor

## 📌 Overview

This project demonstrates a lightweight **multi-container runtime system** integrated with a **kernel-level memory monitoring module**.
It simulates container workloads and tracks system resource usage such as CPU, memory, and I/O in real time.

---

## 🎯 Objectives

* Implement a basic container runtime environment
* Monitor kernel-level memory usage
* Track CPU and I/O activity
* Visualize system performance under load
* Understand OS-level resource management concepts
